### Employee Task Tracker

## Project Description

Employee Task Tracker is a simple web application built using ASP.NET MVC and Entity Framework (Database First).
It allows an admin to assign tasks to employees and employees to view their assigned tasks.

## Technologies Used

ASP.NET MVC
Entity Framework (Database First)
SQL Server
C#
HTML, CSS

## Application Features

Admin login
Employee login
Add tasks for employees
View all tasks (Admin)
View assigned tasks (Employee)
Search tasks by employee
